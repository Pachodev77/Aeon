# Music Folder

This folder contains music files for the MusicPlayer component.

## Supported Formats
- MP3
- WAV
- M4A
- OGG

## How to Add Music
1. Place your music files in this folder
2. The MusicPlayer will automatically detect and load them
3. Songs will appear in the playlist in alphabetical order

## File Naming
For best results, name your files with clear titles:
- `song-name.mp3`
- `artist - song-title.mp3`
